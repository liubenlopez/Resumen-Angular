Pipes

Las pipes en Angular son funciones simples que se utilizan en expresiones de plantilla para aceptar un valor de entrada y devolver un valor transformado. Son útiles para transformar cadenas, cantidades de moneda, fechas y otros datos para su visualización. Se pueden utilizar para transformar la producción y se pueden implementar con tuberías integradas o personalizados.

El texto que se introduce en el siguiente input se le agrega la pipe nativa "uppercase"

{{textToPipe | uppercase}}

El siguiente ejemplo es una pipe personalizada para filtrar la lista de países mediante el contenido del input.
