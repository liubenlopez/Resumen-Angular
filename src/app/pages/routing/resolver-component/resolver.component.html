Resolvers

Los resolutores (Resolvers) en Angular son proveedores de datos que se utilizan para obtener datos antes de que Angular haya cargado completamente el componente asociado. Ayudan a mejorar la experiencia del usuario al proporcionar datos por adelantado, que pueden usarse para completar la plantilla del componente. Los resolver implementan el método resolve(), que se invoca justo antes de que comienza la navegación.

Como la data viene de un resolver hasta q el resolve no tenga la data el componente no se renderiza.
