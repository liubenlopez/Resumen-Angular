Enrutamiento

Angular router-outletes una directiva que actúa como marcador de posición. Angular lo llena dinámicamente según el estado actual del enrutador. Cuando un usuario navega a una ruta específica, el componente correspondiente se carga en el archivo router-outlet. Esto permite la carga dinámica de componentes según la ruta actual. También permite generar un árbol de rutas hijas. Todo esto se define en el módulo app-routing.
