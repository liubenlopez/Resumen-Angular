Query Params

Los parámetros de consulta en Angular se refieren a la capacidad de pasar datos de una ruta a otra dentro de una aplicación de una sola página (SPA). Estos parámetros permiten la transmisión de valores opcionales a una ruta. Al utilizar estos parámetros puede acceder y manipular datos a través de diferentes rutas dentro de una aplicación Angular.

Para usar Query Params en Angular, puede aprovechar la directiva RouterLink y el servicio ActivatedRoute. La directiva RouterLink proporciona opciones para configurar y manejar parámetros de consulta, mientras que el servicio ActivatedRoute ofrece una forma de acceder y observar estos parámetros en los componentes de su aplicación.
