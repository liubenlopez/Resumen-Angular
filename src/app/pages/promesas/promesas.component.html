Promesas

Las promesas son una forma de manejar operaciones asincrónicas. Una promesa es un objeto que representa un valor que puede estar disponible ahora, en el futuro o nunca, como la respuesta a una petición http. Esta representación encapsula el resultado (éxito o error) de una operación asíncrona y el código que debajo de la promesa se sigue ejecutando.

Una promesa en JavaScript se puede encadenar con otras promesas para crear una secuencia de operaciones asíncronas. Esto se conoce como "cadena de promesas" o "promesas en cadena". Cuando una promesa se completa exitosamente, se puede utilizar el método .then() para encadenar la siguiente promesa. El método .then() recibe una función que se ejecutará cuando la promesa se resuelva con éxito y puede recibir el resultado como argumento. Si ocurre un error durante la ejecución de la promesa, se puede utilizar el método .catch() para manejarlo. La cadena de promesas permite ejecutar varias operaciones asíncronas una tras otra, de manera que cada operación se ejecuta después de que la anterior haya completado exitosamente.
